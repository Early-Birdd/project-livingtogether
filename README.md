# project-livingtogether

# 개요
> * 온라인으로 이루어지는 전자 상거래 서비스를 제공하는 웹 사이트이다.
> * 구매자 입장에서의 구매 프로세스, 판매자 입장에서의 상품 관리를 구현할 것이다.

# 구현 기능
> * 권한별 회원가입
> * 로그인
> * 상품 등록 및 관리
> * 주문
> * 장바구니
